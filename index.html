Preface

Welcome!

This book is a guide for translators of rOpenSci materials.

It contain:

  • motivation for multilingual publishing.
  • the general guidelines for translation in English.
  • specific guidelines for translation in each language.
  • the glossary, including the technical term to translate and the technical terms do not translate.

This book is a living document. You can view updates to our best practices and policies via the release notes.

You can cite this book using its Zenodo metadata and DOI.
